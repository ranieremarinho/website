404


"O erro 404 ou 4:04 é um código de resposta HTTP que indica que o cliente pôde comunicar com o servidor, mas o servidor não pôde encontrar o que foi pedido, ou foi configurado para não cumprir o pedido e não revelar a razão, a página não existe mais ou a URL foi inserida incorretamente. Eles não devem ser confundidos com outros erros nos quais uma conexão para o servidor de destino não pôde ser feita." (Wikipédia)
